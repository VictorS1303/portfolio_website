GRUNDLÆGGENDE WEB

Introduktion til det grundlæggende indenfor webudvikling, nemlig HTML og CSS. Herefter introduceredes vi til de to primære layoutmodeller, Flexbox og Grid, hvorefter vi lærte om media queries med henblik på mobile first. Temaet afsluttedes med studiestartsprøven.

Grundlæggende UX/UI

I dette tema blev vi introduceret til forskellige researchtyper, og vi blev indført i, hvad det vil sige at lave en procesdokumentation. Derudover lærte vi at lave prototyper samt en smule om lidt mere avancerede CSS-selektorer og lighthouse tests.

ANIMATION

I dette tema introduceredes vi til begreber som paper prototype, aktivitetsdiagrammer og designprincipper. Ydermere lærte vi også om CSS-positionering og -animationer (keyframes). Derudover lærte vi også om Illustrator til at lave assets til vores spil.

GRUNDLÆGGENDE INDHOLD

Dette temas primære formål var at få os endnu mere indkørt i Premiere Pro samt introduceret os til SCRUM-konceptet og lære os at arbejde efter det ved brug af Trello boards. Derudover blev vi også bekendt med Git, Github og Github Pages samt CSS custom properties og præsentationsteknikken Pecha Kucha.
